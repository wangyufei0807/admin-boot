# 更新日志

所有重要的项目更新都会记录在此文件中。

## [1.0.0] - 2026-05-22

### 初始版本

#### 新增
- 基础项目结构（多模块 Maven 项目）
- RBAC 权限管理系统
  - 用户管理（CRUD、状态管理、密码修改）
  - 角色管理（CRUD、菜单分配）
  - 菜单管理（CRUD、树形结构）
  - 字典管理（字典类型、字典数据）
  - 系统配置管理
  - 操作日志记录
  - 登录日志记录
- JWT 认证系统
  - Access Token + Refresh Token
  - Token 黑名单机制
  - 自动续期
- 文件上传功能
  - 本地存储
  - 支持多种文件类型
- 代码生成器
  - 一键生成 CRUD 代码
  - 支持 DTO/Query/VO
  - 生成 Mapper XML
- API 文档
  - Knife4j 集成
  - Swagger UI 支持

#### 技术栈
- Spring Boot 3.2.5
- MyBatis-Plus 3.5.7
- JWT 0.12.5
- Redis 7.x
- MySQL 8.0

#### 文档
- README.md - 项目说明文档
- architecture-design.md - 架构设计文档
- DEV-GUIDE.md - 开发指南
- DEPLOY.md - 部署指南

## 版本命名规范

本项目使用语义化版本 (SemVer)：
- 主版本号：做了不兼容的 API 修改
- 次版本号：做了向下兼容的功能新增
- 修订号：做了向下兼容的问题修正

## 发布说明

- `✨` 新增功能
- `🐛` 问题修复
- `⚡️` 性能优化
- `📝` 文档更新
- `🔧` 配置变更
- `💥` 破坏性变更
